Your rooms now support beautiful photo backgrounds! Here's how to add them.

## üé® Option 1: Use Unsplash (Free High-Quality Images)

### Step 1: Find a Photo
1. Go to https://unsplash.com
2. Search for: "conference room", "meeting room", "office space"
3. Find a photo you like

### Step 2: Get the URL
1. Click on a photo
2. Right-click the image
3. Select "Copy Image Address" or "Copy Image URL"
4. Example URL:
   ```
   https://images.unsplash.com/photo-1497366216548-37526070297c?w=800
   ```

### Step 3: Add to Room
1. Go to Create Room form
2. Paste URL in "Room Photo URL" field
3. See instant preview!
4. Click "Create Room"

## üîß Troubleshooting

### Photo doesn't show up?
- Make sure URL is publicly accessible
- Check URL ends with image extension (.jpg, .png)
- Try opening URL in a new browser tab
- Check browser console (F12) for errors

### Photo looks stretched or distorted?
- Use images with 16:9 or 4:3 aspect ratio
- Minimum 800px wide recommended
- Use landscape orientation

### Image loads slowly?
- Use smaller file sizes (under 500KB)
- Use image CDNs like Unsplash or Cloudinary
- Consider compressing images before upload
